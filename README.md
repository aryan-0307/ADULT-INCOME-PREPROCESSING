# UCI Adult Income Dataset - Data Preprocessing & Exploratory Data Analysis

## 📌 Overview

This project demonstrates an end-to-end data preprocessing and exploratory data analysis (EDA) workflow on the UCI Adult Income Dataset. The objective is to clean, understand, and prepare the dataset for machine learning models that predict whether an individual's annual income exceeds **$50K**.

---

## 📂 Dataset

- **Dataset:** UCI Adult Income Dataset
- **Task:** Binary Classification
- **Target Variable:** `income`
  - `<=50K`
  - `>50K`

---

## 🚀 Project Workflow

- Imported and explored the dataset
- Identified numerical and categorical features
- Handled missing values using `SimpleImputer`
- Removed duplicate records
- Performed statistical analysis
- Conducted Exploratory Data Analysis (EDA)
- Visualized numerical and categorical features
- Detected and analyzed outliers
- Encoded categorical variables
- Prepared the dataset for machine learning

---

## 📊 Exploratory Data Analysis

The EDA includes:

- Distribution of numerical features
- Frequency distribution of categorical features
- Missing value analysis
- Duplicate detection
- Outlier detection using boxplots
- Statistical summaries
- Feature-wise observations
